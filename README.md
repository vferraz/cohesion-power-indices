# Cohesion-Sensitive Power Indices

Replication package for:

> **Cohesion-Sensitive Power Indices: Representation Results for Banzhaf and Shapley Values**
> Thomas Pitz and Vinicius Ferraz (2026)

This repository provides all materials needed to reproduce the formal verification and
empirical illustrations in the paper.

## Quick start

### Reproduce all figures

```bash
cd figures
uv sync
uv run jupyter notebook cohesion_analysis.ipynb
```

The notebook generates four publication-quality figures (PDF + PNG):

| Figure | Content |
|--------|---------|
| `fig1_apex` | Apex game: power index sensitivity to cohesion parameter b |
| `fig2_bundestag` | Germany 2025: Scenarios A (ideology) and B (cordon sanitaire) |
| `fig3_france_bloc` | France 2024 bloc model: Scenarios A, B, C |
| `fig4_france_party` | France 2024 party model: Scenarios A, B, C |

### Verify proofs

```bash
cd lean4
lake build    # downloads Mathlib + type-checks all proofs
```

Requires [elan](https://github.com/leanprover/elan) (the Lean version manager).
The first build downloads Mathlib and may take 10-30 minutes.

## Data sources

Seat counts come from official election results (Bundeswahlleiterin; Assemblée nationale).
Ideological positions use the `lrgen` variable (0-10 left-right scale) from the
**Chapel Hill Expert Survey (CHES) 2019** (Jolly et al., *Electoral Studies*, 2022).
Bloc-level positions are computed as seat-weighted averages of constituent party scores.

Full source URLs and aggregation methodology are documented in
[`data/SOURCES.md`](data/SOURCES.md).

## Empirical applications

### Germany: Bundestag 2025 (Figure 2)

Seat distribution: CDU/CSU 208, AfD 152, SPD 120, Grüne 85, Linke 64.
Majority threshold: 316/630.
Two scenarios: (A) pure ideology-based cohesion, (B) cordon sanitaire against AfD.

### France: Assemblée Nationale 2024 (Figures 3-4)

Hung parliament following the June-July 2024 snap election.
Analysis at both the bloc level (5 groups) and party level (6 groups).
Three scenarios: (A) pure ideology, (B) cordon sanitaire (RN), (C) double cordon (RN + LFI/NFP).
Key finding: the double cordon at the bloc level produces governance impossibility
(Ensemble + LR + Others = 243 < 289), while party-level analysis reveals a viable
PS-Verts + Ensemble corridor.
